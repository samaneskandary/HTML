# ⚽ ساکر استارز — فوتبال دو بعدی (PWA)

یک بازی فوتبال دو بعدی سبک و **کاملاً آفلاین**، شبیه «Soccer Stars».
بدون هیچ کتابخانه‌ای، فقط HTML/CSS/JS خالص. روی هر نسخه‌ی اندروید و هر مرورگری اجرا می‌شود و قابل نصب مثل اپ است (PWA).

## امکانات نسخه‌ی اولیه
- منوی اول با دکمه‌ی **«بزن بریم»**
- صفحه‌ی **انتخاب زمین** (زمین ۱ / ۲ / ۳) + دکمه‌ی **روشن/خاموش کردن صدا**
- دو تیم **قرمز** (تو) و **آبی** (حریف هوشمند)، زمین، دروازه و تور
- کنترل لمسی: انگشتت را روی یکی از مهره‌های قرمز بکش و رها کن تا شوت بزنی (مثل تیرکمان)
- فیزیک برخورد، اصطکاک، گل و برد/باخت (هر تیم به ۳ گل برسد برنده است)
- اندازه‌ی صفحه **خودکار** تنظیم می‌شود (پرتره/افقی، هر رزولوشنی)
- صدا با Web Audio تولید می‌شود (بدون فایل صوتی، حجم صفر)

## ساختار فایل‌ها
```
soccer-stars/
├── index.html      ← کل بازی (HTML + CSS + JS در یک فایل)
├── manifest.json   ← تنظیمات نصب به‌عنوان اپ
├── sw.js           ← سرویس‌ورکر برای اجرای آفلاین
├── icon-192.png    ← آیکن اپ
└── icon-512.png    ← آیکن اپ
```

## اجرا (روی کامپیوتر برای تست)
چون از سرویس‌ورکر استفاده می‌کند، بهتر است با یک سرور محلی باز شود (نه با دابل‌کلیک):

```bash
cd soccer-stars
python3 -m http.server 8000
```
بعد در مرورگر باز کن: `http://localhost:8000`

> فقط می‌خواهی سریع ببینی؟ همان `index.html` را مستقیم در مرورگر باز کن؛ خود بازی کار می‌کند، فقط حالت «نصب آفلاین» نیاز به سرور دارد.

## انتشار (Deploy) — ساده‌ترین حالت
این پنج فایل را در هر هاست استاتیکی بگذار. همین. هیچ بیلدی لازم نیست:
- **GitHub Pages**: فایل‌ها را در یک ریپو بگذار → Settings → Pages → از شاخه‌ی main منتشر کن.
- **Netlify / Vercel / Cloudflare Pages**: پوشه را Drag & Drop کن.
- یا روی هر سرور/هاست معمولی آپلودشان کن.

نکته: حتماً از **HTTPS** سرو شود تا قابلیت نصب (PWA) فعال شود.

## نصب روی گوشی (به‌عنوان اپ)
آدرس سایت را در مرورگر گوشی باز کن، سپس:
- **اندروید (Chrome)**: منوی سه‌نقطه → «Add to Home screen / افزودن به صفحه اصلی».
- **آیفون (Safari)**: دکمه اشتراک‌گذاری → «Add to Home Screen».

بعد از نصب، تمام‌صفحه و آفلاین اجرا می‌شود.

## (اختیاری) ساخت فایل APK
اگر حتماً APK خواستی، نیازی به بازنویسی نیست؛ همین PWA را با یکی از این‌ها بسته‌بندی کن:
- **PWABuilder** (pwabuilder.com): آدرس سایتت را بده، APK تحویل بگیر.
- **Bubblewrap** (ابزار رسمی گوگل): `npx @bubblewrap/cli init --manifest <آدرس manifest.json>`

## شخصی‌سازی سریع
داخل `index.html` بالای بخش اسکریپت، ثابت‌ها قابل تغییرند:
- `WIN_GOALS` تعداد گل برای برد
- `MAX_SPEED` / `FRICTION` احساس شوت و لغزش
- `FORM` چیدمان مهره‌ها در هر زمین
- `AI` سختی حریف در هر زمین (`spread` کمتر = حریف دقیق‌تر)
